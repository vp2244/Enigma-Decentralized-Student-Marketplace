\textbf{Owner: Member~1.} \texttt{EnigCredit} is an ERC-20 token (name \emph{EnigCredit}, symbol \emph{ENGC}, 18 decimals) deriving from OpenZeppelin \texttt{ERC20}, \texttt{ERC20Burnable}, \texttt{ERC20Permit}, and \texttt{Ownable}. The constructor mints a 1{,}000{,}000 ENGC initial supply to the deployer, who becomes the owner. Issuance is restricted: \texttt{mint(to, amount)} is guarded by \texttt{onlyOwner} so only the faculty/owner account can airdrop demo credits, and non-owner calls revert. Standard \texttt{transfer} and \texttt{approve} provide the value movement and allowances consumed by the escrow slice; \texttt{permit} enables gasless approvals. The front-end token module connects a wallet, displays the live ENGC balance, and exposes the owner-only mint form.
